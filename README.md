# AI Government Transparency Notes

**Can an AI understand government?** An open-source system that measures
how transparent, navigable, and explainable public institutions are, using only
publicly available data.

It reads public documents, budgets, procurement records, audit
reports, and news; produces evidence-backed civic notes; scores six
transparency dimensions with written explanations; and — centrally — records
the civic questions it **could not** answer and exactly what public data was
missing. Missing data is itself a transparency finding.
